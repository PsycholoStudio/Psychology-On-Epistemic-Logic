# 第1章第1節　命題論理の復習 【基本】

## 学習目標

様相論理を学ぶ前に、その基礎となる命題論理について確認します。この節では、命題論理の基本要素である命題変数と論理結合子の理解、真理表による厳密な意味論の習得、妥当な論証と健全な論証の重要な区別の把握、そして心理学研究における論理的推論の中心的重要性について理解することを目標とします。

!!! summary "本節の概要"

    **論理的基盤の確立**：命題変数と5つの論理結合子により、複雑な心理学的関係を厳密に表現する基本的な言語体系を構築する。

    **真理関数的意味論**：真理表による機械的な意味決定により、研究仮説の論理構造を明確化し、実験デザインと直接対応させる。

    **推論の妥当性**：前提から結論への論理的保証を理解し、心理学研究における因果推論や統計的推論の理論的基盤を提供する。

    **心理学的応用**：ストレス研究、認知行動療法、態度変容、発達心理学など具体的領域での形式化により、理論の精密化と検証可能性を実現する。

## 1.1.1 構文と意味論

### 命題論理の基礎理念

様相論理を理解するためには、まずその基礎となる**命題論理（Propositional Logic）**を確実に理解しておく必要があります。命題論理は、アリストテレスの三段論法に遡る古典的伝統を持ちながら、19世紀後半のフレーゲやラッセルによる現代的形式化を経て、今日の数理論理学の基盤となっています。心理学を学ぶ皆さんにとって、論理学は馴染みの薄い分野かもしれませんが、実際には人間の推論プロセスを理解し、研究仮説を厳密に表現するための不可欠な道具となります。

命題論理の根本的なアイデアは、複雑な推論を単純で明確な規則に分解することです。これは心理学における**要素主義**的アプローチと類似しており、複雑な心的現象を基本的な構成要素に分析する研究姿勢と共通しています。ただし重要な違いは、論理学では「真理保存性」、つまり前提が真ならば結論も真でなければならないという厳密な要求があることです。

### 命題変数の性質と心理学的意味

!!! note "定義1.1.1 —— 命題論理の基本要素"

    **命題変数（Propositional Variables）**：\\(p, q, r, s\\) などの文字で表現され、具体的な命題内容を表す基本的な論理単位
    
    **論理結合子（Logical Connectives）**：\\(\\neg, \\land, \\lor, \\rightarrow, \\leftrightarrow\\) により、命題変数を組み合わせて複雑な論理式を構築する演算子

命題論理の基本要素は**命題変数**と**論理結合子**から構成されます。命題変数は \\(p, q, r, s\\) などの文字で表現され、具体的な命題内容を表します。例えば \\(p\\) を「今日は雨が降っている」という命題に対応させることができます。

ここで重要な特徴は、命題変数が**二値性**を持つことです。つまり、命題変数は「真（True）」または「偽（False）」という二つの真理値のどちらか一方を必ず取ります。この二値性は、日常的な人間の思考とは異なる特徴を持っています。

**心理学的考察**：
実際の人間の判断には、「確信がない」「どちらとも言えない」「部分的に正しい」といった**グレーゾーン**が存在します。認知心理学における**ファジィ論理**研究では、こうした曖昧さを \\([0, 1]\\) の連続値で表現しようとします。しかし、古典論理の二値性は、明確な理論的予測と厳密な仮説検証を可能にするという利点があります。

例えば、「被験者は課題を理解している」という命題 \\(p\\) を考えてみましょう。現実には理解度には段階がありますが、実験分析では「理解している/していない」という二分法的判断が求められることが多くあります。この二値化により、統計的検定や因果分析が可能になるのです。

### 論理結合子の体系と心理学的解釈

!!! abstract "定義1.1.2 —— 論理結合子の体系"

    命題論理における5つの基本的論理結合子：
    
    - **否定（\\(\\neg\\)）**：命題の真理値を反転させる単項演算子
    - **連言（\\(\\land\\)）**：複数条件の同時成立を表現する二項演算子  
    - **選言（\\(\\lor\\)）**：複数選択肢の包含的関係を表現する二項演算子
    - **含意（\\(\\rightarrow\\)）**：条件的・因果的関係を形式化する二項演算子
    - **同値（\\(\\leftrightarrow\\)）**：双方向的等価関係を表現する二項演算子

**論理結合子（Logical Connectives）**は、命題変数を組み合わせて複雑な文を作るための道具です。これらは人間の自然言語における接続詞（「そして」「または」「しかし」など）を形式化したものですが、自然言語とは異なる厳密な定義を持っています。以下の5つが基本的な結合子となります：

| 記号 | 読み方 | 意味 | 例 | 心理学的類例 |
|------|--------|------|-----|-------------|
| \\(\\neg\\) | ノット | 否定（〜でない） | \\(\\neg p\\): p でない | 否定的認知、回避行動 |
| \\(\\land\\) | アンド | 連言（〜かつ〜） | \\(p \\land q\\): p かつ q | 複数条件の同時満足 |
| \\(\\lor\\) | オア | 選言（〜または〜） | \\(p \\lor q\\): p または q | 選択肢、代替行動 |
| \\(\\rightarrow\\) | ならば, インプライズ | 含意（〜ならば〜） | \\(p \\rightarrow q\\): p ならば q | 因果関係、条件的推論 |
| \\(\\leftrightarrow\\) | 同値, イフ・アンド・オンリー・イフ | 同値（〜ならば、かつそのときに限り〜） | \\(p \\leftrightarrow q\\): p ならば q、かつ q ならば p | 双方向的因果関係 |

**記号の歴史と意味**：
これらの記号は数学的記法として国際的に標準化されており、心理学研究の国際的な議論においても共通言語として機能します。\\(\\neg\\) は「negation」から、\\(\\land\\) は「logical and」から、\\(\\lor\\) は「logical or」から、\\(\\rightarrow\\) は右向きの矢印で「implies」を、\\(\\leftrightarrow\\) は左右矢印で「if and only if」を表現しています。

**自然言語との違い**：
重要なのは、これらの論理結合子が日常言語の接続詞とは異なる厳密な定義を持つことです。例えば、日常語の「または」は多くの場合**排他的選言**（どちらか一方のみ）を意味しますが、論理の \\(\\lor\\) は**包含的選言**（両方とも可能）を表します。「コーヒーまたは紅茶を飲む」という日常表現では普通どちらか一方を意味しますが、\\(p \\lor q\\) では両方とも真である場合も含みます。

論理記号の習得により、曖昧になりがちな日常語による表現を厳密化し、研究仮説や理論的主張の精度を大幅に向上させることができます。これは特に、**操作的定義**や**仮説の明確化**において重要な役割を果たします。

### 心理学的応用例の体系的分析

論理記号を心理学研究に応用する際の利点を、具体的な研究例を通じて詳しく検討してみましょう。

#### 例1：ストレス研究における多因子モデル

**設定**：ストレスと認知機能の関係を研究する場合を考えます。
- \\(p\\): 「ストレスレベルが高い」
- \\(q\\): 「睡眠の質が悪い」  
- \\(r\\): 「集中力が低下する」
- \\(s\\): 「記憶課題の成績が低い」

**基本的な関係式**：
1. \\(p \\land q\\)：「ストレスが高く、かつ睡眠の質が悪い」（複合的リスク状態）
2. \\((p \\land q) \\rightarrow r\\)：「複合的リスク状態ならば集中力が低下する」（複合仮説）
3. \\(r \\rightarrow s\\)：「集中力低下ならば記憶成績も低下する」（認知機能の般化）
4. \\((p \\land q) \\rightarrow (r \\land s)\\)：「複合リスクは複合的認知機能低下を招く」

**研究上の利点**：
この形式化により、複雑な多因子関係を明確に表現できます。また、実証研究では各命題の真偽を操作的に定義し、統計的に検証することが可能になります。例えば、\\(p\\) は PSS（Perceived Stress Scale）（Cohen et al., 1983）スコア、\\(q\\) は PSQI（Pittsburgh Sleep Quality Index）（Buysse et al., 1989）、\\(r\\) と \\(s\\) は認知課題成績によって測定できます。

#### 例2：認知行動療法における思考パターン分析

**設定**：うつ病の認知的要因を分析します。
- \\(a\\): 「完璧にできない」
- \\(b\\): 「自分には価値がない」
- \\(c\\): 「抑うつ気分が生じる」
- \\(d\\): 「活動を回避する」

**認知的歪みの論理構造**：
1. \\(a \\rightarrow b\\)：「完璧主義的評価」（全か無か思考）
2. \\(b \\rightarrow c\\)：「自己価値と感情の連結」
3. \\(c \\rightarrow d\\)：「感情と行動の連結」
4. \\((a \\rightarrow b) \\land (b \\rightarrow c) \\land (c \\rightarrow d) \\rightarrow (a \\rightarrow d)\\)：「推移性による行動回避」

**治療的含意**：
論理分析により、具体的な介入ポイントが明確になります。認知的介入では、「完璧にできないならば価値がない」という前提（\\(a \\rightarrow b\\)）の妥当性を問うことで、全か無か思考の修正が可能になります。行動的介入では、「完璧にできないならば活動を回避する」という結論（\\(a \\rightarrow d\\)）の自動的連鎖を断ち、段階的な行動活性化を促進できます。感情調節の観点からは、「抑うつ気分ならば回避行動」（\\(c \\rightarrow d\\)）という自動的反応を修正し、感情と行動の間により適応的な関係を築くことができます。

#### 例3：社会心理学における態度変容

**設定**：説得による態度変容を分析します。
- \\(p\\): 「情報源が信頼できる」
- \\(q\\): 「メッセージ内容が論理的」
- \\(r\\): 「受け手が動機づけられている」
- \\(s\\): 「態度変容が生じる」

**精緻化見込みモデルの論理化**（Petty & Cacioppo, 1986）：
1. \\((p \\land q) \\land r \\rightarrow s\\)：「中央ルート処理による態度変容」
2. \\(p \\land \\neg r \\rightarrow s\\)：「周辺ルート処理による態度変容」
3. \\(\\neg p \\rightarrow \\neg s\\)：「信頼性なしでは変容なし」

この形式化により、異なる説得メカニズムの条件と結果を明確に区別し、実験的に検証することが可能になります。

#### 例4：発達心理学における能力獲得

**設定**：ピアジェの保存課題における認知発達を分析します。
- \\(a\\): 「具体的操作期に達している」
- \\(b\\): 「可逆性を理解している」
- \\(c\\): 「保存概念を獲得している」
- \\(d\\): 「系列化課題ができる」

**発達の論理的前提**：
1. \\(a \\rightarrow b\\)：「発達段階と認知操作の対応」
2. \\(b \\rightarrow c\\)：「可逆性理解と保存概念の関係」
3. \\(a \\rightarrow (c \\land d)\\)：「発達段階と複数能力の同時出現」

これらの例から分かるように、論理記号を用いることで重要な利点が得られます。まず、理論の明確化において、仮説の論理構造が明示的に表現されるため、理論の前提と結論の関係が曖昧さなく理解できます。次に、検証可能性の観点から、各構成要素が操作的に定義できるため、実証研究による理論検証が可能になります。また、介入の指針として、論理構造の分析によりどの部分に働きかけることが最も効果的かが明確になり、治療や教育的介入の戦略立案に役立ちます。さらに、理論間比較において、異なる心理学的アプローチを統一的な論理的枠組みで比較検討することが可能になり、学派を超えた理論統合への道筋が開かれます。

## 1.1.2 真理表と意味論

### 真理関数的意味論の基礎

!!! info "定義1.1.3 —— 真理関数的意味論"

    **真理表（Truth Table）**：各論理結合子について、すべての可能な真理値組み合わせに対する出力を表で定義する方法
    
    **真理関数的意味論**：複合命題の真理値が、その構成要素の真理値のみによって機械的に決定される意味論的原理
    
    **合成性の原理**：全体の意味が部分の意味から体系的に決定される言語の基本的性質

命題論理の意味は**真理表（Truth Table）**によって定義されます。これは各論理結合子の意味を、すべての可能な真理値の組み合わせについて表にしたものです。この手法は19世紀後半のフレーゲとピアースによって開発され、現代論理学の基盤となっています。

真理表の本質は**真理関数的意味論**にあります。つまり、複合命題の真理値は、その構成要素の真理値のみによって決定されるという原理です。これは**合成性の原理**とも呼ばれ、言語理解や推論の認知プロセスを説明する際の重要な概念でもあります。

**真理値記号の統一**：

| 記号 | 意味 | 英語 | 数値表現 |
|------|------|------|----------|
| T | 真 | True | 1 |
| F | 偽 | False | 0 |

### 全論理結合子の真理表分析

#### 1. 否定（¬）：認知的反転操作

| \\(p\\) | \\(\\neg p\\) |
|---------|---------------|
| T       | F             |
| F       | T             |

**心理学的意味**：否定は最も基本的な認知操作の一つです。**発達心理学**では、子どもが否定概念を理解するのは2-3歳頃とされており、これは**象徴的思考**の発達と密接に関連しています。臨床心理学では、否定的認知は**うつ病**や**不安障害**の中核的特徴として注目されています。

#### 2. 連言（∧）：条件の同時満足

| \\(p\\) | \\(q\\) | \\(p \\land q\\) |
|---------|---------|------------------|
| T       | T       | T                |
| T       | F       | F                |
| F       | T       | F                |
| F       | F       | F                |

**心理学的意味**：連言は**複数条件の同時満足**を表現します。認知心理学における**注意の分割**研究では、複数のタスクを同時に遂行する能力（\\(p \\land q\\)）の限界が重要な研究テーマとなっています。また、診断基準では複数症状の同時出現（例：DSM-5における大うつ病の診断）で頻繁に使用されます。

#### 3. 選言（∨）：代替可能性

| \\(p\\) | \\(q\\) | \\(p \\lor q\\) |
|---------|---------|-----------------|
| T       | T       | T               |
| T       | F       | T               |
| F       | T       | T               |
| F       | F       | F               |

**心理学的意味**：選言は**代替可能性**や**柔軟性**を表現します。この包含的選言は、日常思考の排他的選言（「AかBかどちらか一方」）と異なり、両方とも可能な場合を含みます。**創造性研究**では、多様な解決策の生成（\\(p \\lor q \\lor r \\lor ...\\)）が**発散的思考**の指標として用いられます。

#### 4. 含意（→）：条件的関係

| \\(p\\) | \\(q\\) | \\(p \\rightarrow q\\) |
|---------|---------|------------------------|
| T       | T       | T                      |
| T       | F       | F                      |
| F       | T       | T                      |
| F       | F       | T                      |

**心理学的解釈の詳細分析**：
含意の真理表は、心理学研究において最も重要でありながら、最も誤解されやすい概念です。特に「前件が偽なら含意は真」という**空虚真理**（vacuous truth）の概念は、直観と対立します。

**科学的仮説との対応**：
「ストレスが高いならば集中力が低下する」（\\(p \\rightarrow q\\)）という仮説を考えてみましょう：
- 行1（T,T）：「ストレス高、集中力低下」→ 仮説支持
- 行2（T,F）：「ストレス高、集中力正常」→ 仮説反証（反例）
- 行3（F,T）：「ストレス低、集中力低下」→ 他の要因による（仮説とは無関係）
- 行4（F,F）：「ストレス低、集中力正常」→ 仮説と整合（他の要因なし）

この分析により、含意の真理表が**反証主義的科学観**（Popper, 1959）と完全に対応していることがわかります。科学的仮説は、それに反する事例（行2）が見つかったときのみ偽とされ、単に合致する事例があるだけでは真とは確定されません。

#### 5. 同値（↔）：双方向的関係

| \\(p\\) | \\(q\\) | \\(p \\leftrightarrow q\\) |
|---------|---------|----------------------------|
| T       | T       | T                          |
| T       | F       | F                          |
| F       | T       | F                          |
| F       | F       | T                          |

**心理学的意味**：同値関係は**双方向的因果関係**や**定義的関係**を表現します。例えば、「自尊心が高い ↔ 抑うつ症状が低い」のような**逆相関関係**や、操作的定義における「IQが130以上 ↔ 知的優秀」のような**定義的同値**で使用されます。

### 真理表と実験デザイン

真理表の各行は、実験における**条件の組み合わせ**に対応します。2要因実験デザインでは、まさに真理表の4行が4つの実験条件となります：

**例：ストレス（有/無）× 社会的支援（有/無）の2×2デザイン**

| ストレス | 社会的支援 | 予測される適応度 | 実験条件 |
|----------|------------|------------------|----------|
| 高（T）  | 有（T）    | 中程度           | 条件1    |
| 高（T）  | 無（F）    | 低               | 条件2    |
| 低（F）  | 有（T）    | 高               | 条件3    |
| 低（F）  | 無（F）    | 中程度           | 条件4    |

この対応により、論理的分析と実験デザインが直接的に結びつき、**交互作用効果**の検出や**媒介分析**の理論的基盤が提供されます。

## 1.1.2 自然演繹とタブロー法

命題論理では、論理的推論を体系化するためのいくつかの証明システムがあります。ここでは**自然演繹**（Natural Deduction）と**タブロー法**（Tableau Method）という二つの主要な方法を紹介します。

### 自然演繹システム

**自然演繹**は、人間の自然な推論過程を形式化した証明システムです。各論理結合子に対して**導入規則**（Introduction Rules）と**除去規則**（Elimination Rules）が定められています。

#### 基本的な推論規則

**連言の規則**：

- 導入規則（∧I）：\\(\\phi\\) と \\(\\psi\\) から \\(\\phi \\land \\psi\\) を導出
- 除去規則（∧E）：\\(\\phi \\land \\psi\\) から \\(\\phi\\) または \\(\\psi\\) を導出

**選言の規則**：

- 導入規則（∨I）：\\(\\phi\\) から \\(\\phi \\lor \\psi\\) を導出
- 除去規則（∨E）：場合分けによる推論

**含意の規則**：

- 導入規則（→I）：\\(\\phi\\) を仮定して \\(\\psi\\) を導けば \\(\\phi \\rightarrow \\psi\\)
- 除去規則（→E）：\\(\\phi \\rightarrow \\psi\\) と \\(\\phi\\) から \\(\\psi\\)（Modus Ponens）

| 記号 | 読み方 | 意味 |
|------|--------|------|
| ∧I | 連言導入 | And Introduction |
| ∧E | 連言除去 | And Elimination |
| →I | 含意導入 | Implication Introduction |
| →E | 含意除去 | Implication Elimination |

### タブロー法

**タブロー法**は、論理式の充足可能性を調べる決定手続きです。証明したい結論の否定を仮定し、矛盾を導くことで間接的に証明を行います。

#### タブロー法の基本手順

1. **否定の仮定**：証明したい式 \\(\\phi\\) の否定 \\(\\neg\\phi\\) から開始することで、間接証明の手法を用います
2. **分解規則の適用**：複合式を構成要素に分解し、より単純な形に変換します
3. **閉鎖の判定**：すべての枝が閉鎖されれば \\(\\phi\\) は妥当であることが証明されます

**分解規則の例**：

- \\(\\neg(\\phi \\land \\psi)\\) → \\(\\neg\\phi \\lor \\neg\\psi\\)（ド・モルガン法則）
- \\(\\phi \\rightarrow \\psi\\) → \\(\\neg\\phi \\lor \\psi\\)（含意の定義）

### 心理学的応用

これらの証明システムは、人間の推論プロセスの研究にも応用できます：

**自然演繹と人間の推論**：自然演繹の規則は、人間の直観的推論に比較的近いとされています。心理学実験では、人々がどの推論規則を自然に使用するかを調べることで、認知的推論メカニズムの解明に貢献できます。

**タブロー法と反証的思考**：タブロー法の「仮定して矛盾を導く」アプローチは、**批判的思考**や**仮説検証**の認知プロセスと類似しています。これは科学的推論における反証主義的アプローチとも関連しています。

**エラーの分析**：推論エラーを、正しい推論規則からの逸脱として分析することができます。例えば、**後件肯定の誤謬**は →E 規則の誤用として理解でき、認知バイアス研究の理論的基盤を提供します。

## 1.1.3 完全性と健全性

このセクションタイトルを正確にします。

!!! warning "定義1.1.4 —— 論証の評価基準"

    **妥当な論証（Valid Argument）**：前提が真である場合に結論も必ず真となる論証の論理的構造
    
    **健全な論証（Sound Argument）**：妥当でありかつ前提が実際に真である論証（論理的正しさ + 事実的正しさ）
    
    **重要な区別**：妥当性は論理的形式の問題、健全性は内容の真理性も含む総合的評価

命題論理では、論証の正当性を評価するための重要な概念があります。**妥当な論証（Valid Argument）**とは、前提が真である場合に結論も必ず真となる論証のことです。一方、**健全な論証（Sound Argument）**とは、妥当でありかつ前提が実際に真である論証を指します。

### 妥当な推論の例

以下は**肯定的後件（Modus Ponens）**と呼ばれる妥当な推論形式の例です：

```
前提1: ストレスが高いならば集中力が低下する (p → r)
前提2: ストレスが高い (p)
結論: 集中力が低下する (r)
```

この推論形式は \\(\\frac{p \\rightarrow q, \\quad p}{q}\\) と表現され、論理的に妥当です。つまり、前提が真であれば結論も必ず真となります。

### 妥当でない推論の例

一方、以下は**後件肯定の誤謬（Affirming the Consequent）**と呼ばれる妥当でない推論です：

```
前提1: ストレスが高いならば集中力が低下する (p → r)
前提2: 集中力が低下している (r)
結論: ストレスが高い (p)
```

この推論は妥当ではありません。集中力低下の原因はストレス以外にも多数存在するからです（睡眠不足、病気、薬物の副作用など）。

### 心理学研究における意義

妥当性と健全性の区別は、心理学研究の質を評価する上で極めて重要です。統計的有意性が得られたとしても、研究デザインや推論の論理構造に問題があれば、科学的に意味のある結論は得られません。

例えば、相関研究において「A と B に相関がある」から「A が B の原因である」と結論するのは、論理的に妥当ではありません。これは前述の後件肯定の誤謬と同様の構造を持っています。因果関係を主張するためには、実験デザイン、統制条件、交絡変数の除去など、より厳密な方法論が必要となります。

### 演繹と帰納

命題論理の推論は**演繹的推論**の一種です。演繹的推論では、前提が真であれば結論も必ず真となります。これに対し、心理学研究でよく用いられる**帰納的推論**では、前提が真であっても結論の真理性は保証されません。

例えば、「これまでの研究で X という介入は効果的だった」から「次回も X は効果的だろう」という推論は帰納的です。過去の成功が未来の成功を論理的に保証するわけではありませんが、合理的な期待を形成する基礎となります。

心理学における理論構築では、演繹的推論と帰納的推論の両方が重要な役割を果たします。既存の理論から具体的な予測を導く過程は演繹的であり、個別の観察から一般的な法則を見出す過程は帰納的です。
